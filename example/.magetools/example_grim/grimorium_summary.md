# Domains
Geolocation, Meteorological Services, User Identity Management

# Summary
This collection provides functional interfaces for retrieving contextual user metadata and localized environmental data. It enables agents to personalize interactions by accessing authenticated profile information and real-time weather conditions based on geographical positioning.

# Major Capabilities
- **User Identity and Geospatial Context**: Facilitates the retrieval of the authenticated user's name and precise geographical coordinates (city, country).
- **Environmental Data Acquisition**: Provides real-time meteorological reporting, supporting both explicit location queries and automated fallback to the user's current detected location.
